## Overview

A capstone data analysis project combining Python, NLP, sentiment analysis and topic modeling. Built to turn large-scale review data into practical business recommendations.

## Approach

- Collected and cleaned large-scale Yelp review datasets
- Applied sentiment analysis to understand customer satisfaction patterns
- Used topic modeling (LDA) to identify recurring themes and pain points
- Created actionable business insights from unstructured text data

## Tech Stack

Python · pandas · scikit-learn · NLTK · Gensim · matplotlib
